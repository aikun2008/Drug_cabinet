### 步骤1: 配置DNS解析
1. 登录域名服务商控制台
2. 添加 DNS 解析记录:
   - A 记录: `@` → `8.134.109.28`
   - A 记录: `www` → `8.134.109.28`

### 步骤2: 配置服务器（云服务器 8.134.109.28）

#### 2.1 安装 Nginx（如未安装）
```bash
# CentOS
sudo yum install nginx

# Ubuntu/Debian
sudo apt install nginx
```

#### 2.2 配置 Nginx 反向代理
创建配置文件 `/etc/nginx/conf.d/drug_cabinet.conf`:
```nginx
server {
    listen 80;
    server_name mguzomb.cn www.mguzomb.cn;

    # Flask 应用反向代理
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # EMQX 相关端口（如需要）
    location /mqtt/ {
        proxy_pass http://127.0.0.1:1883;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }

    location /emqx/ {
        proxy_pass http://127.0.0.1:18083;
        proxy_set_header Host $host;
    }
}
```

启动 Nginx:
```bash
sudo nginx -t
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### 2.3 配置 HTTPS（SSL证书）
使用 Let's Encrypt 免费证书:
```bash
sudo yum install certbot python3-certbot-nginx
sudo certbot --nginx -d mguzomb.cn -d www.mguzomb.cn
```

## 注意事项

1. **ICP备案**: 必须完成 ICP 备案后才能正常使用域名访问
