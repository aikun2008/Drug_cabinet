批量药品入库

×

上传Excel文件批量导入药品信息，同时在现实中向药柜放入对应药品。

支持 .xlsx 和 .xls 格式

Excel文件应包含以下列（顺序不限）：

  • • medicine_code (必填): 药品编码(RFID)
  • • name (必填): 药品名称
  • • type: 药品类型 (0-普通药品, 1-危化品, 2-管制药品)
  • • specification: 规格
  • • manufacturer: 生产厂商
  • • batch_number: 批号
  • • production_date: 生产日期 (YYYY-MM-DD)
  • • expiry_date: 有效期 (YYYY-MM-DD)
  • • storage_condition: 存储条件
  • • location: 所在柜子
  • • unit: 单位
